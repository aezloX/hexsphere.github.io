My Bio

Hexsphere, is a -year-old indie game developer making games in Scratch and graphic art and UI in Affinity Designer. He got his passion for graphic design and coding in Scratch and in HTML from a lifelong love of computers and the things that can be made with them.
He started Scratch in 2013 and has moved only once to his current account Hexsphere. While doing Scratch he has made a AngularJs Order Form, an HTML5 WebGL Tennis Game, and a responsive website he made for a udemy course, and of course, this website.

History Of This Website

After working on Scratch for a while, he decided to broaden his coding knowledge, so he created an HTML website as a side project. Since he already had a Wix website, he didn’t think this would go anywhere, but after seeing Minimalion’s HTML website, he thought it would be fun to renovate that old, badly designed site he had made for fun. So after a couple months of hard work and about 52 GitHub commits later, he published this website for the first time.
