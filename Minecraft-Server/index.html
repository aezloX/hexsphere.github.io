Communication ideas!

My friends and I use an app called Skype to chat while we play on the server. It it's much easier to talk long distance while playing than typing everything we want to say. So since you are going to be joining us, I thought that you could ask your dad if he has a Microsoft Skype account, and if he does could you use it to play with us. If he doesn't have a Skype account, you can ask him to set up Skype for you using his email.
